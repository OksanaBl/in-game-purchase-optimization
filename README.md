# Анализ внутриигровой экономики «Секреты Темнолесья»

Цель проекта — оптимизация монетизации путем анализа характеристик персонажей и их влияния на покупку валюты. 
С помощью SQL определено, что общая доля платящих игроков составляет 17,69%, а выбор расы имеет минимальное влияние на конверсию. 
Выявлено наличие сегмента «китов», значительно увеличивающих средний чек (525.69) относительно медианы (74.86).

# In-game Economy Analysis – "Secrets of Darkwood"
This project aims to optimize monetization by analyzing character traits and their impact on currency purchases. 
Using SQL, it was determined that the overall paying user share is 17.69%, with character race having minimal effect on conversion. 
The analysis identifies a "whale" segment that significantly drives the average purchase (525.69) compared to the median (74.86).
